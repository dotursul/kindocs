# Может ли прибор определять остаточный дисбаланс автоматически?
Остаточный дисбаланс ротора рассчитывается программой прибора **автоматически** по результатам измерения вибрации при установке пробных грузов. 
Для этого используется метод трёх пусков для двухплоскостной балансировки.\n\n
Процедура метода:
1. При первом пуске измеряется исходная вибрация.
2. Затем на ротор ближе к одной из опор устанавливается пробный груз известной массы.
3. Производится второй пуск и измеряются параметры вибрации, которые должны измениться вследствие установки пробного груза.
4. Затем пробный груз в первой плоскости снимается и устанавливается во второй плоскости.
5. Производится третий пуск и измеряются параметры вибрации.
6. Пробный груз снимается, и программа автоматически рассчитывает массы и место (углы) установки уравновешивающих грузов.
\n\nТаким образом, прибор может определять остаточный дисбаланс автоматически.

* Проверить **качество установки датчиков**  
* Увеличить **количество усреднений**  
* Установить датчик фазового угла **под углом 5-10°**

# Какие еще факторы, кроме дисбаланса, могут вызывать вибрацию?
Другие причины:  
* **Износ подшипников**  
* **Несоосность узлов**  
* **Резонансные явления**  
* **Посторонние частицы в роторе**  
* **Вибрация смежного оборудования**

# Как отличить вибрацию, вызванную дисбалансом, от вибрации, вызванной другими причинами?
**Критерии отличия:**  
* Вибрация от дисбаланса - **на частоте вращения ротора**  
* Для анализа других причин используется:  
  - **Спектральный анализ**  
  - **Гармонический анализ**  
* Требуется **анализ частотно-амплитудных характеристик**
